# Color to Hex 转换为 Hex 颜色

Convert a color to hex. The alpha value of the color is discarded.

将颜色转换为十六进制。颜色的透明度值将不再使用。

### Color 颜色

A color to convert.

需要转换的颜色。

### Hex 十六进制

The hex representation of color as a text string.

表示十六进制色值的文本字符串。
