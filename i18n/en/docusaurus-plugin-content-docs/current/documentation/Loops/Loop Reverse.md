# Loop Reverse 循环反转

Reverse order of values in a loop.

反转循环中值的顺序。

Right-click to change the type of the loop.

右键单击以更改循环的类型。

![Image](@site/static/img/docs/Loops/loop-reverse.png)

### Loop 循环

A loop to reverse values for.

需要反转值顺序的循环。

### Output 输出

The resulting loop with values in reverse order.

反转值顺序后的结果循环。
