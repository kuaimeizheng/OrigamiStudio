#### Input 输入

##### Spacing Token 间距样式

默认定义好了常用的间距比例，比如 Base (4)、2x (8)、3x (12)、4x (16) 等。

#### Output 输出

##### Spacing 间距

间距数值，可以链接到对应的图层属性上。
