Transform the case and capitalization of a text string.

转换文本字符串的大小写和大小写。

A text string to transform.

要转换的文本字符串。

**Transform 转换**

The property to control capitalization of the text string – Uppercase, Lowercase, or Capitalized.

控制文本字符串大小写的属性——大写、小写或首字母大写。

**Output 输出**
