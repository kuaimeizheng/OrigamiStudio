# Running Total 累积总计

Calculate the sums of a loop of numbers, where the sum at each index is the sum of the numbers preceding the current number.

计算数字循环的总和，其中每个索引处的总和是当前数字之前的数字的总和。

### Input 输入

A loop of numbers, indices, or booleans.

数字、索引或布尔值的循环。

### Output 输出

A loop of sums.

一个循环的总和。
