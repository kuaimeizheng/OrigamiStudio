# Visual Effect 毛玻璃效果

An iOS visual effect view that blurs all the layers below it.

iOS视觉效果视图，模糊下面的所有图层。

**Enable 启用**

A boolean that is true when the layer is displayed.

**Opacity 不透明度**

The opacity of the layer.

**Style 样式**

The style of the blur: light, very light, or dark.

模糊的样式：浅色、非常浅、深色。
