# JSON to Text JSON转文本

Formats any JSON value into a text string to be used with a Text Layer for debugging.

将任何JSON值格式化为一个文本字符串，用文本图层进行测试。

### Input 输入

A JSON object or array.

JSON对象或数组。

### Output 输出

A text string that represents the JSON data.

一个文本字符串，表示JSON数据。
