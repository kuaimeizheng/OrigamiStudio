# Number Formatter

功能：

5个同时数字输入

四舍五入到n位小数

分隔数千

在末尾插入符号

插入+表示正值

带有美元、欧元、英镑、日元和卢布符号的货币格式

短格式，短+货币格式

拼写数字

支持负值
