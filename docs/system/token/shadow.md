# Shadow 阴影

---

快捷设置交互效果的阴影样式。

![](http://ssy-design.oss-cn-shenzhen.aliyuncs.com/images/ixd/patches/shadow.png)

#### Input 输入

##### Shadow Token 阴影样式

设置阴影样式的层级。包括：None、Z 100、Z 200、Z 300、Z 400.

##### Enable 启用

是否启用阴影样式，不启用则为 None。

#### Output 输出

##### Shadow Opacity 阴影不透明度

阴影的不透明度。

##### Shadow Radius 阴影半径

阴影的半径。

##### Shadow Offset Y 阴影 Y 轴位移

阴影的 Y 轴方向的位移。
