# Layer Interaction 图层交互

---

快捷的设置图层的交互样式。

![](http://ssy-design.oss-cn-shenzhen.aliyuncs.com/images/ixd/patches/interaction.png)

##### 背景类型

可以选择是在 白色背景 或者 灰色背景 上的交互样式。

##### 灰色背景

可以设置在灰色背景上面的默认、悬停、点击的交互样式。

##### 白色背景

可以设置在白色背景上面的默认、悬停、点击的交互样式。

##### 动画

可以设置交互的时候，是否有水波动画效果。
