# Absolute Value 绝对值

Turn any value into a positive value (ex: -40 → 40, 500 → 500, 0 → 0).

Right-click to change the type (ex: number, position, rotation).

将任何值转换为正值 。

右键更改输入接口接入值类型（例如：数字，坐标，角度）。
