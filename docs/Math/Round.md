### Input

The number to round.

输入的数字。

### Rounded 四舍五入

The number rounded to the nearest integer.

数字四舍五入为最接近的整数。

例如：4.3 → 4、4.7 → 5。

### Rounded Down 向下取整### 

The number rounded down.

数字向下取整。

例如：4.3 → 4、4.7 → 4。

### Rounded Up 向上取整### 

The number rounded up.

数字向上取整。

4.3 → 5、4.7 → 5。
