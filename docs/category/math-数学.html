Math 数学

📄️ + 加

Add two or more values together, or join two or more text strings.

📄️ - 减

Subtract one or more values from a base value.

📄️ Absolute Value 绝对值

Turn any value into a positive value (ex: -40 → 40, 500 → 500, 0 → 0).

📄️ Arctangent 反正切函数

Calculate the angle of a triangle given its Y and X dimensions.

📄️ Cosine 余弦函数

Calculate the X dimension of a triangle with a hypotenuse of 1.

📄️ Length 长度

Calculate the length of a number, point, point 3D, or vec4, where the length is the square root of each dimension squared (ex: calculate the hypotenuse of a triangle with a point X and Y).

📄️ Math Expression 数学表达式

Compute any mathematical expression. Edit the expression by right-clicking the patch and choosing “Patch Info” from the menu. Include variables in the expression to define the inputs to the patch. Any math expression that is valid in JavaScript is supported. Use functions from the JavaScript Math library to write more complicated expressions. See Math Expressions for examples and more information.

📄️ Max 最大值

Compare two or more values and output the largest one. Although the patch’s default type is number, it can be used to compare position, point, color and index values as well.

📄️ Min 最小值

Compare two or more values and output the smallest one. Although the patch’s default type is number, it can be used to compare position, point, color and index values as well.

📄️ Mod 求余函数

Calculate the remainder when one or more values are divided.

📄️ Power 幂值

Calculate the first value (base) to the second value (exponent)’s power (ex: 2^3).

📄️ Round 四舍五入

Round a number.

📄️ Sine 正弦

Calculate the Y dimension of a triangle with a hypotenuse of 1.

📄️ √ 根号

Calculate the square root of a value.

📄️ x 乘

Multiply two or more values.

📄️ ÷ 除

Divide a value by one or more values.
