Layer 图层

📄️ Color Fill 颜色填充

Fill the screen with a solid color. Clip color fills with groups set to a particular size.

📄️ Gradient Fill 渐变填充

Fill the screen with a linear gradient of two colors. The gradient’s start and end positions can be controlled to create vertical, horizontal or diagonal gradients. Clip color fills with groups set to a particular size.

📄️ Group 组

Organize related layers that move, scale, rotate, and fade together. Groups have a size that clips the layers within.

📄️ Hit Area 点击热区

Add interaction to a specific rectangle on the screen. Toggle the ‘Setup Mode’ of the hit area to see where it’s positioned.

📄️ Image File 图像文件

Select an image to add as an Image Layer.

📄️ Image Layer 图像图层

A layer that can display an image.

📄️ Live Image 实时图像

A layer that always displays the latest version of an image file on the computer. When the selected image is updated the image of this layer will automatically be updated to the latest version.

📄️ Map 地图

Shows a static snapshot of a map using Apple Maps API.

📄️ Oval 椭圆形

An oval of specific size and fill color.

📄️ Particle System 粒子系统

A particle emitter. Configure the specific birthrate, velocity, angle and color change over their lifetime. Add an image to replace the standard circular particle.

📄️ Progress Ring 进度圈

Create a customizable circular progress layer.

📄️ Rectangle 矩形

A rectangle of a specific size and fill color.

📄️ Shape 形状

A shape specified by the shape input. Add a shape patch like Rectangle, Triangle, Oval, or Union of the above. Shapes can have a color fill and a stroke.

📄️ Text Layer 文本图层

Display text. The alignment, size, font, and all other standard attributes of text can be configured on this layer.

📄️ Video File 视频文件

Select a video to add as a Video Layer.

📄️ Video Keyframes 视频关键帧

A layer that can play output of the Keyframes Exporter as a video. Keyframes is a library for converting Adobe After Effects shape based animations to a data format that allows rendering on iOS and Android devices.

📄️ Video Layer 视频图层

A layer that can play a video.

📄️ Video Stream 视频流

A layer that can play a video stream using HTTP Live Streaming.

📄️ Viewfinder 取景器

Say cheese! A live stream of the current computer, phone, or tablet’s camera. ‘Freeze’ the viewfinder to prototype taking a photo.
