Loops 循环

📄️ Loop 循环

Create a loop of indices, ex: a count of 3 outputs a loop of [0, 1, 2]. Similar to a for loop in programming.

📄️ Any 任意

Check if any of the booleans in a loop are true.

📄️ Grid Layout 网格布局

Repeat a layer or group in a list or grid. Evenly divides the total grid width by the number of columns to quickly size and position the items.

📄️ Loop Builder 循环生成器

Create a loop of any value (ex: text, color, image).

📄️ Loop Count 循环数量

Count the number of values in a loop.

📄️ Loop Dedupe 循环重复数据删除

Remove duplicate values in a loop.

📄️ Loop Filter 循环筛选

This is a very powerful patch that can do a lot of things! It can filter items out of a loop, but it can also repeat items in a loop. It can even be used to create a new loop with a single value repeated X times.

📄️ Loop Insert at End 循环结尾插入

Insert a value at the end of a loop.

📄️ Loop Insert 循环插入

Insert a value at a particular index of a loop.

📄️ Loop Option Switch 循环选项开关

Find the index of the last pulsed item in a loop of pulses.

📄️ Loop Over Array 数组遍历

Create a loop over the items in an array.

📄️ Loop Remove Last 循环末尾移除

Remove the last value from a loop.

📄️ Loop Remove 循环移除

Remove a value from a particular index of a loop.

📄️ Loop Reverse 循环反转

Reverse order of values in a loop.

📄️ Loop Select 循环选择

Select one or more values from a loop.

📄️ Loop Shuffle 环随机排序

Randomly reorder values in a loop.

📄️ Loop Sum 循环总和

Calculate the sum of every value in a loop.

📄️ Loop to Array 循环转换数组

Convert a loop to an array.

📄️ Running Total 累积总计

Calculate the sums of a loop of numbers, where the sum at each index is the sum of the numbers preceding the current number.
