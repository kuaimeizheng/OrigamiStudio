Device 设备

📄️ Browser Buttons 浏览器按钮

Captures clicks on the back and forward buttons when using the Browser Chrome patch.

📄️ Browser Chrome 浏览器

Style the viewer window to look like a web browser. Use the Browser Buttons patch to capture interactions.

📄️ Camera 摄像头

Say cheese! A live stream of the current computer, phone, or tablet’s camera that supports recording video and capturing images.

📄️ Device Buttons 设备按钮

Access the buttons on a phone or tablet. Currently supports the Android back button.

📄️ Device Info 设备信息

Access information about the simluated or connected phone, tablet or computer.

📄️ Device Motion 设备运动

Access accelerometer and gyroscope information from the phone, tablet, or computer.

📄️ Device Time 设备时间

Access the time on the connected phone, tablet, or the computer Origami is running on.

📄️ Game Controller 游戏控制器

Control your prototype using any MFI-compatible Game Controller. Supports the Game Pad and Extended Game Pad profiles. Micro Game Pad profile is not available on macOS.

📄️ Haptic 触觉

Access the haptic engine on an iPhone or iPad.

📄️ Interface Orientation 界面方向

Define how the prototype’s interface reacts to rotation of the device. Different phones, laptops or tablets can have different set of supported interface orientations. Use the Device Info patch to get the physical orientation of a connected device in angles.

📄️ Location 位置

Gets the current location of the device (using its GPS) and returns coordinates in the form of latitude and longitude. Connect to a Map Layer to visualize the location.

📄️ Microphone 话筒

Record sound from the phone, tablet, or computer microphone. Connect to a Sound Player to play the recorded sound.

📄️ Mouse Cursor 鼠标手势

Control the system mouse cursor.

📄️ Sound Kit 音效包

Play from a collection of UI sound effects. For expanded descriptions and documentation, visit the Facebook Design Resources.

📄️ Sound Player Setting 声音播放器设置

Customize a Sound Player.

📄️ Sound Player 声音播放器

Play any sound or music file.

📄️ Touches 触摸

Capture the touches from the simulated or connected phone, tablet or computer.

📄️ Trackpad Haptic 触控板触觉

Access the haptic engine on a macOS trackpad.

📄️ Trackpad 触控板

Capture touches on the computer trackpad.

📄️ Vibrate 震动

Vibrate the connected phone or tablet.
